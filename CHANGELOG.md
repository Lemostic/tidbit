# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 进行版本管理。

## [0.1.1] - 2026-07-13

### 新增

- 新增简体中文与英文界面语言，可在设置中即时切换；翻译资源统一存放于 `resources/locales`，便于社区贡献更多语言。
- 新增后台自动加密备份，可按 0.5 小时步长设置 0.5–24 小时间隔，并设置保留 1–10 份自动备份。
- 全面更新应用、任务栏与托盘图标，提供高分辨率主图标和针对小尺寸优化的托盘图标。
- 主界面新增应用置顶功能；置顶后即使窗口吸附到屏幕边缘也不会自动隐藏。
- 便签列表新增全局排序，支持按便签名称、创建时间、更新时间及手动顺序排列。
- 手动排序模式新增专用拖动手柄，可直接拖放便签卡片并持久化顺序。
- 面板便签列表和云游便签卡片新增一键复制，并可在设置中选择 Markdown 或保留列表编号与层级的格式化纯文本。
- 全局颜色管理新增一组鲜艳、高辨识度的现代配色。

### 优化

- 优化 Markdown 编辑器与便签卡片的列表缩进、段落间距和内容渲染。
- 液态玻璃与普通界面透明度现可调至完全不透明，并增强旧版 WebView 环境的样式回退能力。
- 新建分组时自动从上一分组颜色之外的候选色中选择随机颜色。
- 优化窄窗口下搜索框的文字布局与响应式表现。
- 分组区域在液态玻璃等模式下始终保持透明。
- 提升新增分组输入框在不同主题和透明度下的文字可读性。

### 修复

- 修复 Tauri/WebView 客户端中手动排序模式无法可靠拖动便签卡片的问题。
- 修复主面板文字可被浏览器式选中的问题，同时保留编辑器与输入框的正常文本选择能力。
- 修复部分旧版内网环境中液态玻璃效果只显示黑白边线、透明底色不可见的问题。
