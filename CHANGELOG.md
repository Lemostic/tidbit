# Changelog

本文件记录 tidbit 的用户可见变更。版本格式遵循语义化版本，尚未发布的功能统一归入 `Unreleased`。

## [Unreleased]

### 新增

- 便签编辑器现在支持录制语音备忘录。点击工具栏中的麦克风即可开始录音，录制期间可以查看时长、停止并插入或取消。
- 录音会作为独立内容块保存在便签中，支持播放、删除和修改名称；默认名称为开始录音时的本地时间戳。
- 主便签列表和云游便签均可直接播放录音，操作播放器不会误触发便签编辑。

### 安全与质量

- 仅允许安全的音频 Data URL 进入便签渲染，并继续移除脚本、自动播放及事件处理属性。
- 增加录音插入、重命名、持久化、播放器交互、内容安全策略与 MediaRecorder 端到端验证。

## [0.1.2] - 2026-07-15

### 新增

- 设置中新增 Windows 开机自动启动开关，默认关闭，可随时启用或停用。

### 工程

- 完善 Windows 安装包构建配置、代码检查和自动启动相关测试。
